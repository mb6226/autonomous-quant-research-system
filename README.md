# Autonomous Quant Research System (AQRS)

Autonomous Quant Research System is a research platform for discovering,
evaluating, optimizing and ranking quantitative trading strategies.

## Vision

Build a system that can:

- Collect market data
- Generate features
- Train models
- Validate strategies
- Rank experiments
- Optimize parameters
- Discover new trading approaches

without manual intervention.

- DJI

## Architecture

Market
→ Dataset Factory
→ Feature Factory
→ Target Factory
→ Model Factory
→ Experiment Runner
→ Validation Engine
→ Backtest Engine
→ Research Database
